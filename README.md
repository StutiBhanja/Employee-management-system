# Employee Management System 🗄️

## Overview
A normalized relational database system built using MySQL to manage employee data across departments.

## Tools Used
![MySQL](https://img.shields.io/badge/MySQL-005C84?style=flat&logo=mysql&logoColor=white)
![SQL](https://img.shields.io/badge/SQL-orange?style=flat)

## Database Structure
6 interlinked tables:
- **Employee** — personal and login details
- **JobDepartment** — job roles and departments
- **SalaryBonus** — monthly, annual salary and bonus
- **Qualification** — employee certifications
- **Leaves** — leave tracking
- **Payroll** — central payment calculation table

## Key Highlights
- 21 analytical SQL queries across 5 categories
- Foreign key constraints with CASCADE and SET NULL
- Salary expenditure analysis
- Department-wise headcount reporting
- Top-earner identification
- Leave pattern analysis
- Payroll compensation analysis
